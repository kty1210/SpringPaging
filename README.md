<!-- ABOUT THE PROJECT -->
## 소개
<br/>
<br/>



페이지 화면을 구현한 게시판

<br/>

화면에 출력되는 글 개수를 제한해서 가시성 향상

<br/>

최신글일수록 역순으로 보기 쉽게 정렬

<br/>
<br/>
<br/>
<br/>




#### 주요 구현 메소드
<br/>

* save : 새 글 등록
* findAll : 게시글 조회
* findById : 게시글 상세 조회
* delete : 게시글 삭제
* update : 게시글 수정
* paging : 요청 페이지 출력


<br/>
<br/>

#### 추가된 라이브러리
<br/>

* lombok
* JUnit
* MyBatis
* HikariCP
* log4j2
